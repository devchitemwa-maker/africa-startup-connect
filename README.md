
# Africa Startup Connect (ASC)

Africa Startup Connect (ASC) is a platform that connects entrepreneurs, investors,
and skilled partners to build real businesses in Africa, starting with Zambia.

## Features
- Landing Page
- Signup Page (Demo)
- Dashboard (Demo)
- Single-file HTML (easy hosting)

## How to use
Upload `index.html` to any hosting or open it locally in a browser.
